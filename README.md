# Take-home-Martin-Cheong-

Bank Statement Parser created by Martin Cheong. 

## 1. Clone the repository

## 2. Get API Key
Please go to https://developer.pdf.co/

Register for an account, then go to the home page to get your API key under "Your API Key" (the free version gives you 10000 credits, which is more than enough for 100 API calls)

## 3. Set API Key
Create an .env file, and then set REACT_APP_API_KEY to your API key (Please see .env.example)

## 3. Install dependencies
npm install

## 4. Run the web server
npm run start

Thank you!
